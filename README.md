Backend  
---
[Backend](https://github.com/bailongctui/Backend)是一个基于**NodeJS** + **Express** + **MongoDB**的后台管理系统demo。  

此项目的主要模块有：
* bin       启动脚本目录
* model  数据表配置目录
* public  公共文件
* route    路由目录
* views  模板目录
* app.js 配置主文件
* package.json 安装模块信息  

环境要求
---
1.  Npm
2.  Ruby
3.  Express
4.  MongoDB
5.  ...
